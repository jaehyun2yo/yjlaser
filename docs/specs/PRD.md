# PRD — 유진레이저목형 통합 웹 플랫폼

## 제품 비전

도무송 목형 제작 전문 업체의 업무 프로세스를 디지털화하여, 거래처 소통부터 납품까지 전 과정을 웹에서 관리한다.

## 사용자 유형

| 사용자      | 설명                        | 접근 경로   |
| ----------- | --------------------------- | ----------- |
| 관리자      | 유진레이저 내부 직원        | /admin/\*   |
| 거래처      | 도면을 의뢰하는 B2B 고객사  | /company/\* |
| 현장 작업자 | 레이저가공/오시칼 현장 직원 | /worker/\*  |
| 일반 방문자 | 웹사이트 방문자             | / (public)  |
