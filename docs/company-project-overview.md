# 회사 프로젝트 종합 현황

작성일: 2026-06-17
참조형 관리 기준 추가: 2026-06-18
전수조사 반영: 2026-06-18

## 목적

이 문서는 유진레이저목형의 활성 프로젝트가 각각 어떤 기능을 담당하고, 어디까지 진행됐으며, 어떤 프로그램/서비스와 연동되는지 한눈에 보기 위한 상위 현황판이다. 세부 최신 상태를 직접 소유하지 않고, 각 프로젝트의 원천 문서로 연결하는 참조형 색인 역할을 한다.

코드 실행이나 운영 재검증 없이 저장소 안의 README, CLAUDE, roadmap, todo, changelog, 기존 상태 문서를 근거로 작성했다. 진행률은 실제 품질 점수나 배포 안정성 점수가 아니라 `docs/roadmap.md` 체크박스 완료율이다.

## 참조형 관리 원칙

- 이 문서는 전체 조망, 우선순위, 리스크, 원천 문서 위치를 보여주는 문서다.
- 최신 기능 목록, 최근 수정사항, 배포 변경사항, 세부 이슈는 각 원천 문서에 기록한다.
- 같은 사실을 여러 문서에 길게 복사하지 않는다. 상세 내용은 원천 문서에 두고 이 문서는 링크와 요약만 유지한다.
- 기능/상태가 바뀌어도 이 문서는 큰 역할, 연동, 위험도, 우선순위가 바뀔 때만 수정한다.
- 문서 간 상태가 다르면 이 문서에서 임의로 확정하지 않고 `전수조사 반영 결과`에 조치/남은 확인으로 올린다.

## 회사 자동화 큰그림

```text
거래처
  ├─ 웹사이트 문의/도면 업로드
  │    -> yjlaser_website
  │    -> 자체 웹하드 / Cloudflare R2
  │
  └─ LGU+ 외부웹하드 업로드
       -> 외부웹하드동기화프로그램
       -> 자체 웹하드 / Cloudflare R2

최종 도면/DXF 작업
  -> 네트워크 DXF 작업폴더
  -> 유진레이저목형 관리프로그램
       - 업체명/가격 파싱
       - 업체별/월별 분류
       - Excel 청구서 생성
       - Popbill FAX/MMS 또는 Email 발송
  -> 레이저네스팅프로그램
       - DXF 합판 배치
       - 결과 DXF 생성
       - 레이저가공 Excel 작업목록 기록

작업/납품 관리
  -> yjlaser_website Worker/ERP
       - 관리자/현장 작업관리
       - 거래처 주문/작업 상태 확인
       - 납품 흐름 관리

운영 인프라
  -> computeroff
       - 사무실/현장 PC 부팅/종료/하트비트 모니터링
```

## 공통 리스크

- 외부 서비스 의존: LGU+ 웹하드, Popbill, Supabase, R2, Railway, Vercel 상태에 영향을 받는다.
- 민감 정보: API Key, Popbill/SMTP 자격증명, 거래처 정보, 고객 도면은 문서와 로그에 값이 남지 않아야 한다.
- 문서 정합성: 이번 전수조사에서 확인된 주요 표현 불일치는 갱신했지만, 기능 구현 완료와 운영 검증 완료는 계속 분리해서 기록해야 한다.
- 운영 검증 공백: 기능 구현 완료와 실제 현장 장기 운용 검증은 분리해서 봐야 한다.
- 데이터 경계: 웹, 데스크톱 앱, 네트워크 폴더, Excel, SQLite/Supabase 사이의 소유권과 동기화 정책을 계속 명확히 해야 한다.
